1 Install Certificate

Your private CA is stored at {{.CertPath}}.

It cannot be used to issue certificates for legacy domains (ending with .com, .net ... etc) since it uses the name constraints extension. Add this CA to your browser/TLS client trust store to allow Fingertip to issue certificates for decentralized names.
